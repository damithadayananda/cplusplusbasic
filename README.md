Data types

c++ offer following data types

Boolean(bool)
Character(char)
Integer(int)
Floating point(float)
Double floating point(double)
Valueless(void)
Wide character(wchar_t)

above types can be modified using following modifiers

signed
unsigned
short
long

constant(literals)

1)#define id = 5;
2)const char NEWLINE = '\n';
